# Design Guidelines — Avisignal

## Colors
- Primary background: Deep navy (#0A1628)
- Surface/cards: Dark slate (#1E293B)
- Accent: Electric cyan (#00D4FF)
- Success/positive signals: Green (#22C55E)
- Danger/negative: Red (#EF4444)
- Warning: Amber (#F59E0B)
- Text primary: White (#F8FAFC)
- Text secondary: Muted (#94A3B8)

## Typography
- Clean modern sans-serif (Inter or similar)
- Data-forward: metrics and numbers large and prominent
- Monospace font for signal values, accuracy percentages, timestamps

## UI Style
- Dark theme — data dashboard aesthetic
- Mobile-first, optimized for Android screen sizes
- Signal prediction cards with confidence score badges
- Accuracy gauges and percentage meters
- Smooth, real-time animated trend charts
- Clean data tables for signal history log
- Minimal chrome, maximum data density

## Key Components
- Hero signal card (current prediction + confidence %)
- Accuracy gauge (overall hit rate)
- Trend chart (historical + projected curve)
- Signal history feed (scrollable table)
- Team stats summary bar
